# 🚀 Personal Command Center

A modern personal productivity dashboard built using HTML, CSS, and JavaScript.  
It helps users manage tasks, notes, and daily workflow in one place.

---

## ✨ Features
- Digital clock with 12h/24h toggle and optional seconds
- Live date with multiple format options
- Weather widget with humidity and wind speed
- Daily Focus with streak tracking
- Pomodoro timer with customizable durations
- To-do list with priorities and filters
- Quick notes with autosave and download
- Daily habits tracker
- Motivational quotes with copy function
- Productivity statistics
- Weekly activity chart
- Productivity score calculation
- Dark/Light theme with accent colors
- Background animations toggle
- Keyboard shortcuts
- Data export/import/reset
- Full responsive design
- Glassmorphism UI

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
